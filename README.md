# PTI AR

## Overview
Custom Odoo 18 module for PTI AR functionality.

## Installation
1. Place this module in your Odoo addons path.
2. Update the module list: **Settings > Apps > Update Apps List**.
3. Search for "PTI AR" and install.

## Configuration
No special configuration required.

## Usage
After installation, access the module from the PTI AR menu.

## Dependencies
- `base`
